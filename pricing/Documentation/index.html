A) HTML Structure - top

We make the HTML Structure so simple as we can.
How to switch plan type: 

	<div class="plan-container plan-easy green span12">
Notice the key "class" plan-easy is using to change pricing type
Available classes type:
  • plan-easy
  • plan-cool
  • plan-old-man
